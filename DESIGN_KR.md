# Coralithm 디자인 설명

## 현재 제품 상태

Coralithm은 선택한 관심사를 3D 알고리즘 산호로 바꾸는 인터랙티브 웹 경험입니다. 사용자는 관심 카테고리 3개를 선택하고, 산호를 생성한 뒤 카테고리 비중을 조절하고, 산호 태그를 변경하고, 산호를 삭제하고, 해당 산호와 연결된 추천 콘텐츠 피드를 볼 수 있습니다.

현재 기본 웹 흐름은 다음과 같습니다.

1. Intro 화면
2. 관심사 선택 화면
3. 산호 생성 / 상세 화면
4. 산호 피드 패널
5. 산호 추가 흐름
6. 전체 산호 모아보기 화면

## Intro 화면

Intro 화면은 어두운 수중 3D 배경과 떠 있는 산호 오브젝트로 구성됩니다. 제목은 이미지가 아니라 웹 텍스트로 표시됩니다.

화면에 보이는 요소:

- `Start your exploration with`
- `CORALITHM`
- `당신의 취향을 선택하고 '알고리즘 산호'를 만들어보세요.`
- `그 구조를 기반으로 다양한 콘텐츠를 추천해드릴게요.`
- `시작하기`

`시작하기`를 누르면 관심사 선택 화면으로 이동합니다.

## 관심사 선택 화면

관심사 선택 화면에서는 사용자가 카테고리 3개를 선택합니다.

화면에 보이는 문구:

- `어떤 산호를 성장시킬까요?`
- `관심있는 주제 3가지를 선택해주세요.`
- `선택한 주제를 기반으로 산호가 성장합니다.`
- `산호 생성하기`

현재 카테고리는 14개입니다.

- 뉴스·시사
- 경제·시장·투자
- 요리
- 게임
- 스포츠
- 소프트웨어·데이터·AI
- 환경·기후
- 광고·마케팅
- 음악
- 디자인·예술
- 여행
- 영감·인사이트
- 학습
- 스타일

카테고리 3개를 선택해야 `산호 생성하기` 버튼을 누를 수 있습니다. 선택 순서는 초기 비중 50%, 30%, 20% 순서로 사용됩니다.

## 산호 생성 / 상세 화면

산호가 생성되면 화면 중앙에 3D 산호가 표시됩니다. 왼쪽 패널에는 산호 제목, 태그 변경, 비중 슬라이더, 삭제 버튼이 표시됩니다.

화면에 보이는 조작 요소:

- `← 전체 산호`
- 산호 제목, 예: `요리 · 소프트웨어·데이터·AI · 스타일 산호`
- `태그 변경`
- 카테고리 선택 드롭다운 3개
- 비중 슬라이더 3개
- `산호 삭제`
- `+ 산호 추가`
- `피드 보기`

산호 제목은 현재 적용된 카테고리 태그 3개를 조합해 표시됩니다. 산호 상세 화면의 상단에도 현재 산호 제목이 표시됩니다.

## 태그 변경

`태그 변경` 영역에서는 현재 산호에 연결된 세 개의 카테고리 태그를 바꿀 수 있습니다.

현재 동작:

- 각 태그 위치는 카테고리 드롭다운으로 구성됩니다.
- 다른 위치에서 이미 선택된 카테고리는 드롭다운에서 비활성화됩니다.
- 태그를 바꾸면 산호 제목이 바로 바뀝니다.
- 태그를 바꾸면 비중 슬라이더의 카테고리 이름이 바로 바뀝니다.
- 태그를 바꾸면 피드 내용이 바로 바뀝니다.
- 태그를 바꾸면 산호의 카테고리 기반 시각 매핑이 바로 바뀝니다.

## 비중 조절

세 개의 슬라이더는 현재 산호의 카테고리 비율을 조절합니다.

현재 동작:

- 세 비율의 합은 항상 100%입니다.
- 음수값이 나오지 않습니다.
- 슬라이더 허용 범위를 넘지 않습니다.
- 한 슬라이더를 움직이면 나머지 비율이 다른 두 카테고리에 분배됩니다.
- 비중을 바꾸면 산호 시각 표현과 피드가 함께 갱신됩니다.

## 산호 삭제

`산호 삭제` 버튼을 누르면 현재 산호가 삭제됩니다. 삭제 후 화면은 전체 산호 모아보기 화면으로 돌아갑니다.

## 산호 피드 패널

`피드 보기`를 누르면 오른쪽에 산호 피드 패널이 열립니다.

피드 패널에 보이는 요소:

- `산호 피드`
- 카테고리 비중 요약
- 콘텐츠 카드 6개
- 오른쪽 위 `×` 닫기 버튼

각 콘텐츠 카드는 다음 정보를 가집니다.

- 출처 / 플랫폼 이름
- 카테고리 라벨
- 콘텐츠 제목
- 이미지 미리보기
- 구체적인 콘텐츠 링크

YouTube 카드는 해당 영상의 썸네일을 사용합니다. 웹페이지 카드는 해당 페이지 URL에서 생성한 미리보기 이미지를 사용합니다. 카드를 클릭하면 해당 콘텐츠 페이지가 새 탭에서 열립니다.

피드 카드는 현재 산호의 세 카테고리에서 선택됩니다. 비중이 높은 카테고리는 더 많은 카드를 받습니다.

## 전체 산호 모아보기 화면

전체 산호 모아보기 화면에서는 사용자가 만든 산호초를 볼 수 있습니다.

화면에 보이는 요소:

- `나의 알고리즘 산호초`
- `산호를 선택하면 취향의 구조와 추천 콘텐츠를 볼 수 있어요.`
- 생성된 산호들
- 산호 라벨
- 산호 개수
- `+ 산호 추가`

산호를 클릭하면 해당 산호의 상세 화면으로 이동합니다. `+ 산호 추가`를 누르면 관심사 선택 화면을 다시 사용해 산호를 하나 더 추가합니다.

## 3D 장면

현재 3D 장면은 다음 요소로 구성됩니다.

- 검은 수중 배경
- bloom 효과
- fog
- vignette
- 떠다니는 입자
- marine snow 입자
- 빛줄기
- orbit 카메라 조작
- 산호 자동 배치
- 산호 성장 애니메이션
- 전체 산호 모아보기 화면의 산호 라벨
- 가까운 산호 사이의 연결선

현재 프로젝트는 7개의 산호 모델을 불러옵니다. 주 카테고리는 산호 색과 기본 모델을 결정합니다. 카테고리 비중은 크기, 투명도, 거칠기, 발광, 형태 변형에 반영됩니다.

## 현재 데이터 매핑

| 입력 | 현재 효과 |
|---|---|
| 선택한 카테고리 3개 | 산호 태그, 제목, 피드 카테고리 |
| 선택 순서 | 초기 50%, 30%, 20% 비중 |
| 주 카테고리 | 산호 색과 기본 모델 |
| 비중 슬라이더 | 카테고리 비율, 산호 외형, 피드 분배 |
| 태그 변경 | 카테고리 태그, 제목, 피드, 시각 매핑 |
| 삭제 버튼 | 현재 산호 삭제 |

## 현재 피드 데이터

피드는 큐레이션된 콘텐츠 라이브러리를 사용합니다. 각 카테고리는 구체적인 콘텐츠 항목을 가지고 있습니다. 카드는 검색 결과가 아니라 특정 웹페이지나 영상으로 연결됩니다.
