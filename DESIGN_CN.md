# Coralithm 设计说明

## 当前产品状态

Coralithm 是一个 3D 互动网页体验。用户选择三个兴趣类别后，系统生成一株算法珊瑚。用户可以调整类别权重、修改珊瑚标签、删除珊瑚，并查看与这株珊瑚相关的推荐内容 Feed。

当前默认网页流程是：

1. Intro 首页
2. 兴趣选择页
3. 珊瑚生成 / 详情页
4. 珊瑚 Feed 面板
5. 追加珊瑚流程
6. 珊瑚总览页

## Intro 首页

首页是深色水下 3D 场景，背景中有漂浮的珊瑚对象。标题使用网页文字实现，不使用截图图片。

页面可见内容：

- `Start your exploration with`
- `CORALITHM`
- `당신의 취향을 선택하고 '알고리즘 산호'를 만들어보세요.`
- `그 구조를 기반으로 다양한 콘텐츠를 추천해드릴게요.`
- `시작하기`

点击 `시작하기` 进入兴趣选择页。

## 兴趣选择页

兴趣选择页要求用户选择三个类别。

页面可见内容：

- `어떤 산호를 성장시킬까요?`
- `관심있는 주제 3가지를 선택해주세요.`
- `선택한 주제를 기반으로 산호가 성장합니다.`
- `산호 생성하기`

当前共有 14 个类别：

- 新闻·时事
- 经济·市场·投资
- 料理
- 游戏
- 体育
- 软件·数据·AI
- 环境·气候
- 广告·营销
- 音乐
- 设计·艺术
- 旅行
- 灵感·洞察
- 学习
- 风格

必须选择三个类别后，`산호 생성하기` 才可点击。选择顺序会成为初始权重顺序：50%、30%、20%。

## 珊瑚生成 / 详情页

生成后，页面中央显示 3D 珊瑚。左侧面板显示珊瑚标题、标签修改、权重滑杆和删除按钮。

页面可见控件：

- `← 전체 산호`
- 珊瑚标题，例如 `요리 · 소프트웨어·데이터·AI · 스타일 산호`
- `태그 변경`
- 三个类别下拉框
- 三个权重滑杆
- `산호 삭제`
- `+ 산호 추가`
- `피드 보기`

珊瑚标题由当前三个类别标签组合而成。珊瑚详情页顶部也显示当前珊瑚标题。

## 标签修改

`태그 변경` 区域用于修改当前珊瑚的三个类别标签。

当前行为：

- 每个标签位使用一个类别下拉框。
- 已经被其他标签位使用的类别会在下拉框中禁用。
- 修改标签后，珊瑚标题立即更新。
- 修改标签后，权重滑杆的类别名称立即更新。
- 修改标签后，Feed 内容立即更新。
- 修改标签后，珊瑚的类别视觉映射立即更新。

## 权重修改

三个滑杆控制当前珊瑚的类别比例。

当前行为：

- 三个百分比始终合计为 100%。
- 不会出现负数。
- 不会超过滑杆允许范围。
- 移动一个滑杆时，剩余百分比分配给另外两个类别。
- 修改权重后，珊瑚视觉和 Feed 会同步更新。

## 珊瑚删除

点击 `산호 삭제` 会删除当前珊瑚。删除后界面返回珊瑚总览页。

## 珊瑚 Feed 面板

点击 `피드 보기` 会在右侧打开 Feed 面板。

Feed 面板可见内容：

- `산호 피드`
- 类别权重摘要
- 六张内容卡片
- 右上角 `×` 关闭按钮

每张内容卡片包含：

- 来源 / 平台名
- 类别标签
- 内容标题
- 图片预览
- 具体内容链接

YouTube 卡片使用对应视频缩略图。网页卡片使用对应页面 URL 生成的预览图。点击卡片会在新标签页打开该具体内容页面。

Feed 卡片来自当前珊瑚的三个类别。权重更高的类别会分配到更多卡片。

## 珊瑚总览页

总览页显示用户已经生成的珊瑚礁。

页面可见内容：

- `나의 알고리즘 산호초`
- `산호를 선택하면 취향의 구조와 추천 콘텐츠를 볼 수 있어요.`
- 已生成的珊瑚
- 珊瑚标签
- 珊瑚数量
- `+ 산호 추가`

点击珊瑚会进入该珊瑚的详情页。点击 `+ 산호 추가` 会复用兴趣选择页，并向珊瑚礁追加一株新珊瑚。

## 3D 场景

当前 3D 场景包含：

- 黑色水下背景
- Bloom 发光
- 雾效
- 暗角
- 漂浮粒子
- 海雪粒子
- 光束
- 轨道相机控制
- 珊瑚自动摆放
- 珊瑚生长动画
- 总览页珊瑚标签
- 邻近珊瑚连接线

项目当前加载 7 个珊瑚模型。主类别决定珊瑚颜色和基础模型。类别权重影响尺寸、透明度、粗糙度、发光和形态变化。

## 当前数据映射

| 输入 | 当前效果 |
|---|---|
| 三个选择类别 | 珊瑚标签、标题、Feed 类别 |
| 选择顺序 | 初始 50%、30%、20% 权重 |
| 主类别 | 珊瑚颜色和基础模型 |
| 权重滑杆 | 类别比例、珊瑚外观、Feed 分配 |
| 标签修改 | 类别标签、标题、Feed、视觉映射 |
| 删除按钮 | 删除当前珊瑚 |

## 当前 Feed 数据

Feed 使用策展内容库。每个类别都有具体内容项。卡片不是搜索入口，而是指向具体网页或视频。
